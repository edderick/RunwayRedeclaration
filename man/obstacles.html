Obstacles

Obstacles represent the obstructions on the runway; they are modeled as a single point, with a height, on, or near, the runway.

Loading an Obstacle

Loading a Saved or Default Obstacle

To load an obstacle, select File (ALT+f) and then Open (ALT+o) followed by obstacle (Alt+o). A file chooser will appear - initially it shall be in the default directory where predefined Obstacles are stored. It is possible to chnage this directory and open an obstacle from anywhere on the computer.

Loading a Recently Used Obstalce

To load a recently used obstacle, click File (ALT+f) and the Open Recent (ALT+r), select Obstacles (ALT+o) and finally select the obstacle from a list of recent obstacles.

Creating an Obstacle

If there isn't already an obstacle that meets your requirements, it is possible to create a new one. Click File (ALT+f) and then click New (ALT+n), finally Obstacle (ALT+o). This will open the edit obstacle dialog, from here the obstacle can be defined

Saving an Obstacle

To save an obstacle, click File (ALT+f). Next press Save (ALT+s), finally select Obstacle (ALT+o). This will bring up a file chooser dialog, choose a place to save the file and click save.

Note: .xml will be appended to the file name that you enter.

Placing Obstacle

To alter the placing or height of an obstacle, click Edit (ALT+e) and then obstacle (ALT+o).
