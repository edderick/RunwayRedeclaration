Getting Started

This software has been designed to be easy to use effectively and efficiently. Most functionality can be accessed through menu items

To begin redeclaring runways, you will need to load in a predefined runway, or create a new one. This can be done from the menus. Click on the appropriate navigation item for more information about this.

Next you will need to create or load an obstacle (again more information in the appropriate manual page). The addition of an obstacle to the runway will automatically update all of the calculations.

More information about the visual representations created by this software can be found elsewhere in the manual.
