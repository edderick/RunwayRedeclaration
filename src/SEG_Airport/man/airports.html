Airports

Airports contain runways, and a few other airport specific parameters

Loading an Airport

When an airport is loaded in all of the runways associated with that airport will be loaded in, and can then be selected to place obstacles on.

Loading a Saved or Default Airport

To load an airport, select File (ALT+f) and then Open (ALT+o) followed by Airport (Alt+a). A file chooser will appear - initially it shall be in the default directory where predefined Airports are stored. It is possible to change this directory and open an airport from anywhere on the computer.

Loading a Recently Used Airport

To load a recently used airport, click File (ALT+f) and the Open Recent (ALT+r), select Airport (ALT+a) and finally select the airport from a list of recent airport.

Creating an Airport

If there isn't already an airport that meets your requirements, it is possible to create a new one. Click File (ALT+f) and then click New (ALT+n). This will open the edit airport dialog, from here new runways can be created, edited and deleted.

Saving an Airport

To save an airport, click File (ALT+f). Next press Save (ALT+s), finally select Airport (ALT+a). This will bring up a file chooser dialog, choose a place to save the file and click save.

Note: .xml will be appended to the file name that you enter.
