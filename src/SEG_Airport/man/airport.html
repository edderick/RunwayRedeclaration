Loading an Airport

When an airport is loaded in all of the runways associated with that airport will be loaded in, and can then be selected to place obstacles on.

Loading a Saved or Default Airport

To load an airport, select File (ALT+f) and then Open (ALT+o) followed by (Alt+a). A file chooser will appear - initially it shall be in the default directory where predefined Airports are stored. It is possible to chnage this directory and open an airport from anywhere on the computer.

Loading a Recently Used Airport

To load a recently used airport, click File (ALT+f) and the Open Recent (ALT+r) and finally select the airport from a list of recent airport.
